# Presentation des données


Nous avons décidé de faire notre projet d'ANDO23 sur l'étude de l'utilisation des Pokémon dans le monde compétitif suivant leur données dans les jeux.

## Le contexte

La franchise Pokémon est une franchise qui depuis 1995 publie des jeux centrés sur la capture de monstres, leur élevage et le combat entre eux. Elle est aujourd'hui la franchise médiatique la plus lucrative au monde, avec près de 150 milliards de USD en revenus.

De nombreuses compétitions de combat entre Pokémon sont apparues, et avec elles des classements d'utilisation des différents Pokémon dans le monde compétitif professionnel. Ces compétitions ont une forte popularité à échelle mondiale, et peuvent se diviser en de nombreuses catégories différentes suivant les Pokémon utilisés et les modalités des combats.

Or tous les Pokémon sont d'attribus, formes, capacitées et types différents, à travers de données codées dans le jeu et publiquement accessibles. Nous avons donc cherché à étudier les liens et corréllations qu'il pourrait exister entre les taux d'utilisation des Pokémon dans le domaine compétitif, et leur données et capacités dans les jeux vidéos.

## Le sujet

Notre sujet est donc : 
- **Study of Pokemon's competitive usage according to game data.**

## Les données

Nous avons décidé de travailler à partir de deux bases de données : 
- Une base de données regroupant toutes les données des Pokémon directement dans le jeu
- Une base de données regroupant les pourcentages d'utilisation des pokémons dans les combats "solo" compétitifs

La première base de donnée regroupe toutes les données qui définissent un Pokémon. Celles-ci recouvrent leurs charactéristiques liées à leur physique (taille, poids, genre, vitesse de croissance...) et celles liées à leur capacités de combat ('type', 'stats', 'EV'...).

Pour choisir la deuxième base de donnée nous avons décidé de nous limiter à l'étude d'un type de combat spécifiquement, les combats "solo", qui consistent en l'affrontement de deux joueurs qui utilisent 6 Pokémon chacun. Dans la catégorie "solo", plusieurs sous-catégories éxistent, qui divisent tous les Pokémon par taux d'utilisation. Nous avons donc décidé de tous les rassembler dans une catégorie, tout en gardant une répartition et un classement équivalents à la répartition actuelle.
La deuxième base de donnée regroupe donc une grande quantité de Pokémon classés par catégorie et taux d'utilisation dans les combats "solo"
